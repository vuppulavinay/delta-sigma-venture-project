

# Frontend
change directory to `frontend` to run frontend 
first install packages -- `npm install`
After that to run project using command -- `npm run start` in local (`http://localhost:3000`)
If you want to see host url --- `https://rainbow-licorice-ae8da8.netlify.app`




# Backend
change directory to `backend` to run backend 
first install packages -- `npm install`
After that to run project using command -- `npm run start` in local (`http://localhost:5000`)
